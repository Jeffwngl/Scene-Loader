# Scene Loader

A web renderer for 3d scenes to test unique shaders quickly and effortlessly, either write your own shader or use the existing ones.

## Shader Passes

The scene will be rendered initially with the THREE JS base shader which can be toggled and manipulated via the side panel, then any additional shader passes are done as a post processing pass through the `EffectComposer`.
